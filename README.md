A Python-based personal finance management application with data visualization capabilities.

## ✨ Features

- 📊 **Transaction Management**: Add, view, and track income and expenses
- 📈 **Data Visualization**: Interactive pie charts and bar graphs
- 💾 **Persistent Storage**: All data saved in JSON format
- 📅 **Monthly Summaries**: Automatic calculation of income, expenses, and savings
- 🏷️ **Category Tracking**: Organize transactions by customizable categories

## 🛠️ Technologies Used

- Python 3.x
- matplotlib (for data visualization)
- JSON (for data storage)

## 📦 Installation

1. Clone this repository:
```bash
git clone https://github.com/abhivarshithachowdary3-sys/Personal-Finance-Tracker.git
cd p6_finance-tracker
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python p6_finance_tracker.py
```

## 💡 Usage

The application provides an interactive menu with the following options:

1. **Add Income** - Record income transactions
2. **Add Expense** - Record expense transactions
3. **View All Transactions** - Display transaction history
4. **Monthly Summary** - View current month's financial overview
5. **Expense Breakdown Chart** - Visualize spending by category (pie chart)
6. **Income vs Expenses Chart** - Compare monthly income and expenses (bar chart)
7. **Exit** - Save and exit the application
